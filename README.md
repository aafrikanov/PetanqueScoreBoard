# Score Server (петанк)

Сервер счёта для отображения и управления счётом двух команд в петанке.

## Возможности

- **Виджет** (`/`, `/widget`) — отображение счёта, названий команд и количества шаров (1–6) у каждой команды.
- **Админка** (`/admin`) — изменение названий команд, счёта (0–13), количества шаров; кнопки «Новая игра» (сброс счёта и шаров) и «Новый гейм» (шары = 6).

Состояние хранится в памяти сервера и сохраняется через API.

## Запуск

```bash
npm install
```

**Разработка** (сервер API + Vite):

```bash
npm run dev
```

Приложение: http://localhost:5173  
API: http://localhost:3001  

**Сборка и production:**

```bash
npm run build
npm run start
```

Сервер отдаёт API и статику из `dist`. Порт задаётся переменной `PORT` (по умолчанию 3001).

## API

- `GET /api/score` — текущее состояние счёта.
- `PUT /api/score` — обновление (тело: `{ team1: { name, balls, score }, team2: { name, balls, score } }`).

## Стек

- Frontend: React, Vite, TypeScript, React Router
- Backend: Express, CORS
